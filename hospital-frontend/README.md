# 医院挂号系统前端

## 技术栈
- Vue 3 + TypeScript
- Element Plus
- Vite
- Pinia
- Vue Router
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构
```
src/
├── api/          # API接口
├── views/        # 页面组件
├── router/       # 路由配置
├── App.vue       # 根组件
└── main.ts       # 入口文件
```
