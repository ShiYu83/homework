##  用户手册

### 系统概述

**系统功能**：

- 在线预约挂号
- 医生排班查询
- 就诊队列管理
- 病例和处方管理
- 费用支付结算
- 药品管理
- 数据统计分析

**访问方式**：

- 前端地址：http://localhost:3000
- 后端API：http://localhost:9090/api
- 开发环境：本地部署

### 患者操作指南

#### 注册登录

**首次使用 - 患者注册**

1. 访问系统首页，点击导航栏中的"注册"按钮
2. 进入注册页面，填写以下信息：
   - 姓名（必填）
   - 身份证号（必填，用于实名认证）
   - 手机号（必填）
   - 性别（必填）
   - 出生日期（可选）
   - 过敏史（可选）
3. 点击"注册"按钮提交信息
4. 注册成功后，系统会提示注册成功

**登录系统**

1. 访问系统首页，点击导航栏中的"登录"按钮
2. 进入登录页面，输入：
   - 用户名（必填）
   - 密码（必填）
3. 点击"登录"按钮
4. 登录成功后，系统会自动跳转到首页
5. 登录后，导航栏会显示"退出"按钮，点击可退出登录

**注意**：
- 如果还没有账号，可以点击登录页面的"还没有账号？立即注册"链接跳转到注册页面
- 登录状态会保存在浏览器中，关闭浏览器后需要重新登录
- 如果后端登录接口尚未配置，系统会使用模拟登录功能（仅用于开发测试）

####  预约挂号

**步骤详解**：

1. **选择就诊方式**
   - 按科室挂号：不适症状明确时使用
   - 按医生挂号：有指定医生时使用
   - 智能导诊：不确定科室时使用
2. **选择科室/医生**
   - 浏览科室列表
   - 查看医生简介和专长
   - 查看患者评价
3. **选择时间**
   - 查看可预约日期
   - 选择具体时间段
   - 确认号源状态
4. **确认预约**
   - 核对预约信息
   - 选择支付方式
   - 完成支付
5. **就诊准备**
   - 查看预约详情
   - 获取就诊提醒
   - 准备相关证件

####  预约管理

**查看预约记录**

- 在"我的预约"中查看所有记录
- 按状态筛选（待支付、已预约、已完成、已取消）
- 查看预约详情（包括病例和处方信息）

**取消预约**

- 已预约状态可以取消
- 需要填写取消原因
- 取消后自动释放号源
- 频繁取消可能影响信用度

**修改预约**

- 已预约状态可以修改
- 可以选择新的日期和时段
- 修改时自动释放原号源并占用新号源

**支付订单**

- 待支付状态的预约需要支付挂号费
- 支持微信、支付宝、医保三种支付方式
- 支付成功后预约状态自动更新为"已完成"
- 已完成状态的预约只显示"查看详情"按钮，不能修改或取消

**查看病例和处方**

- 医生看病后可以查看病例信息（症状、诊断、医嘱）
- 可以查看处方详情（药品列表、总金额）
- 处方待支付时可以支付药品费
- 支持合并支付（挂号费+药品费）

### 医生操作指南

#### 登录系统

1. 访问系统首页，点击"登录"按钮
2. 使用医生账号登录（如：doctor001 / 123456）
3. 登录成功后，导航栏会显示医生相关菜单

#### 查看个人信息

1. 点击"我的信息"菜单
2. 查看个人基本信息（姓名、职称、专长等）
3. 查看排班信息和号源统计

#### 排班管理

1. 点击"调班管理"菜单
2. 查看个人排班列表
3. 申请调班/停诊
   - 选择需要调整的排班
   - 填写调班原因
   - 选择新的日期和时段（停诊时可不选）
   - 提交申请
4. 查看调班申请状态（待审核、已通过、已拒绝）

#### 就诊管理

1. **查看患者队列**
   - 查看当日预约的患者列表
   - 支持按日期查询
   - 显示患者基本信息（姓名、性别、手机号、过敏史等）
   - 显示预约状态和预约时间

2. **叫号功能**
   - 点击"叫号"按钮叫号患者
   - 叫号后患者状态更新
   - 可以开始看病流程

3. **看病开药**
   - 填写诊断信息（症状描述、诊断结果、医嘱建议）
   - 添加药品到处方
   - 设置药品数量
   - 提交诊断和处方

4. **查看病例**
   - 看病完成后可以查看已创建的病例
   - 查看患者信息、病例详情、处方信息
   - 支持查看历史病例记录

### 管理员操作指南

#### 登录系统

1. 访问系统首页，点击"登录"按钮
2. 使用管理员账号登录（admin / 123456）
3. 登录成功后，导航栏会显示管理员控制台菜单

#### 医生管理

1. 点击"医生管理"菜单
2. 查看医生列表（支持按科室筛选）
3. 创建新医生：填写姓名、科室、职称、专长等信息
4. 编辑医生信息：修改职称、专长、简介等
5. 删除医生：删除不需要的医生记录

#### 患者管理

1. 点击"患者管理"菜单
2. 查看患者列表（支持关键词搜索）
3. 创建新患者：填写患者基本信息
4. 编辑患者信息：修改患者资料
5. 查看患者详情：包括预约记录、诚信度等

#### 科室管理

1. 点击"科室管理"菜单
2. 查看科室列表
3. 创建新科室：填写科室名称、编码、描述
4. 编辑科室信息
5. 删除科室

#### 诊室管理

1. 点击"诊室管理"菜单
2. 查看诊室列表（支持按科室筛选）
3. 创建新诊室：填写诊室编号、名称、所属科室、位置等
4. 编辑诊室信息
5. 删除诊室

#### 号源管理

1. 点击"号源管理"菜单
2. 查看排班列表（支持多条件筛选）
3. 创建新排班：选择医生、日期、时段、设置号源数量
4. 编辑排班：修改号源数量、状态等
5. 删除排班

#### 药品管理

1. 点击"药品管理"菜单（在管理员控制台）
2. 查看药品列表
3. 创建新药品：填写药品名称、价格、规格、单位、库存、分类等
4. 编辑药品信息
5. 上架/下架药品：控制药品的可用状态
6. 删除药品

#### 调班审核

1. 点击"调班审核"菜单
2. 查看待审核的调班申请
3. 审核申请：通过或拒绝
4. 填写审核意见

#### 数据统计

1. 点击"统计分析"菜单
2. 查看预约统计：总预约数、待支付、已预约、已取消、已完成数量
3. 查看排班统计：总排班数、总号源数、已预约数、剩余号源、号源利用率
4. 支持按时间范围筛选

### 常见问题解答

**Q: 忘记密码怎么办？**
A: 在登录页点击"忘记密码"，通过手机验证码重置。

**Q: 预约成功后如何就诊？**
A: 按预约时间提前15分钟到达医院，在自助机或人工窗口取号。

**Q: 可以帮家人预约吗？**
A: 可以，在"家庭成员"中添加家人信息后即可代为预约。

**Q: 挂号费可以退款吗？**
A: 未就诊情况下，在规定时间内取消预约可自动退款。

**Q: 支付后预约状态会变化吗？**
A: 支付成功后，预约状态会自动更新为"已完成"，此时只能查看详情，不能再修改或取消。

**Q: 医生如何查看患者病例？**
A: 在患者队列页面，如果患者已有病例记录，会显示"查看病例"按钮，点击即可查看完整的病例和处方信息。

**Q: 可以同时支付挂号费和药品费吗？**
A: 可以，在处方详情页面点击"支付订单（挂号费+药品费）"按钮，可以创建合并订单一次性支付。
